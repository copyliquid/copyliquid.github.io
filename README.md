# Copyliquid — HL 跟单雷达

输入任意 Hyperliquid 地址，在浏览器里现场完成：

- **全周期跟单回测**：$100k 起、动态等比镜像该地址的每笔 perp 成交（含 TWAP 切片），taker 4.5bp + 滑点 3bp，资金费逐小时结算，输出净值曲线 / 回撤 / Sharpe / 月度收益
- **交易画像**：TWAP/Maker 占比、持仓集中度、回合胜率盈亏比、持仓时长、活跃时段、入场后 24h 择时胜率
- **标签**：巨鲸 / 金库 / 机构式执行 / 量化高频 / 主观重仓 / 现货对冲 / 疑似信息优势 / 已不活跃
- **可跟单判定**：结合模拟收益、回撤、杠杆、集中度、数据完整度给出结论与理由

纯静态页面（仓库根目录），直连 Hyperliquid 公开 info API（CORS 开放），无后端、无数据落库。

## 已知数据限制

- TWAP 切片明细 API 只保留约最近 2 个月；更早时段用官方 PnL 按权益等比换算拼接（图中有分界标注）
- 普通成交 API 最多返回 1 万笔，超出的最早期历史会截断
- K 线每个周期只保留约 5000 根，早期用日线标记
- 现货成交不复制；未模拟强平与跟单延迟

## 本地研究管线

`scripts/` 下是等价的 Python 离线管线（拉数 → 回测 → 出静态报告），改 `ADDR` 后依次运行。

仅供研究，不构成投资建议。
